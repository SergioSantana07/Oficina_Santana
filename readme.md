# Auto Peças e Mecânica Santana

Sistema web desenvolvido em Django para gerenciamento de oficina mecânica.

## Funcionalidades

- Site institucional
- Produtos
- Serviços
- Contato
- Dashboard administrativo
- Cadastro de clientes
- Cadastro de veículos
- Ordem de serviço

## Tecnologias

- Python
- Django
- Bootstrap 5
- SQLite
